FAQs

  • Q: Do I need a profile to create an event?
  • A: Yes, a profile account must be created before any event can be created or joined.

  • Q: How many events can I attend?
  • A: We allow our users to attend as many events as they want.

  • Q: Why are the zip code for some events "00000"?
  • A: "00000" is the zip code used for online games and virtual meetings.

  • Q: How do I edit my profile once it is created?
  • A: First, make sure you are logged in. Once you are, you can access your profile page from the "Profile" tab on the navbar. Once you see your information, find the "Edit Profile" button. Make any changes and click "Apply".

  • Q: How do I edit my event once it is created?
  • A: Navigate to the event in which you created. You can find a button labeled "Edit Event", which allows you to individually edit the information.

  • Q: Can I share my location with other?
  • A: Currently we don't have this feature, but we will be adding it in a future update.

  • Q: Do I need to pay for any features?
  • A: We are providing this web application free to everyone.

  • Q: Does anything change due to Covid-19?
  • A: There are no limitation due to Covid-19 to use this application. However, this application is used nationwide and you should check with your local officials for any mandates given and follow guidelines by the CDC.
